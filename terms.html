La Saga Manager — Last updated: February 19, 2026

1. About This App

La Saga Manager is a content management tool used exclusively by the Uruguayan rock band La Saga (@lasagarock) to publish videos and updates to TikTok and other social media platforms.

2. Usage

This application is not publicly available. It is used solely by authorized members of La Saga's management team to post content to the band's official TikTok account.

3. Content

All content published through this app is owned by La Saga. The app does not collect, store, or process any third-party user data.

4. TikTok Integration

This app uses the TikTok Content Posting API to publish videos directly to the band's TikTok profile. It requires authorization from the account owner via TikTok's OAuth flow.
